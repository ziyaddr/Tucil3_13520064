## Cara run
    1. Pada folder bin, jalankan main.exe
    2. Pada tampilan utama, tekan "Generate Random Puzzle" untuk mengacak puzzle
    3. Pada tampilan utama, tekan "Read From File" untuk menerima input file
    4. tekan "Solve" untuk menyelesaikan Puzzle secara otomatis
    5. Tombol atas, bawah, kiri, dan kanan pada keyboard jg dapat ditekan untuk menggeser puzzle
